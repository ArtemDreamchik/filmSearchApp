# Film Search App — Поиск фильмов

Веб-приложение для поиска фильмов по названию с отображением подробной информации о каждом из них. Данные загружаются в реальном времени через [OMDB API](https://www.omdbapi.com/).

## ✨ Функциональность

- **Поиск по названию** — вводишь название фильма и нажимаешь «Найти», получаешь список результатов
- **Карточка фильма** — клик по результату открывает детальную страницу с информацией:
  - Постер
  - Название, год выхода, дата релиза
  - Рейтинг, продолжительность, жанр
  - Режиссёр, сценаристы, актёры
  - Описание сюжета
- **Навигация** — кнопка «Назад к поиску» возвращает к списку результатов

## 🔑 Настройка API-ключа

Для работы приложения нужен бесплатный API-ключ от OMDB.

1. Зарегистрируйся на [omdbapi.com](https://www.omdbapi.com/apikey.aspx) и получи ключ
2. В файле `index.js` замени значение переменной с ключом:

```js
const API_KEY = 'твой_ключ_здесь';
```

## 🚀 Запуск

```bash
# Клонируй репозиторий
git clone https://github.com/ArtemDreamchik/filmSearchApp.git

# Перейди в папку проекта
cd filmSearchApp
```

Добавь API-ключ в `index.js`, затем открой `index.html` в браузере.

> **Рекомендуется** запускать через локальный сервер (например, [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) в VS Code), чтобы избежать проблем с CORS при запросах к API.
